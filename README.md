# 🌱 CarbonWise

CarbonWise is an AI-inspired Carbon Footprint Tracker that helps individuals understand, track, and reduce their carbon emissions through personalized recommendations and sustainability challenges.

## Features

- Carbon Footprint Calculation
- Green Score
- Personalized Action Plan
- Weekly Green Challenges
- Sustainability Insights

## Tech Stack

- Python
- Google Colab
- Gradio

## Problem Statement

Many people want to live sustainably but do not understand how their daily habits contribute to carbon emissions.

## Solution

CarbonWise calculates monthly carbon emissions based on transportation, electricity consumption, diet, shopping habits, and plastic usage. It then provides personalized recommendations to help users reduce their environmental impact.

## Impact

Encourages eco-friendly habits and raises awareness about climate change through simple and actionable insights.
